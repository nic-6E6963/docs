# Getting Started with Meridian

This guide will walk you through integrating your financial institution with the Meridian network and completing your first cross-border settlement.

## Prerequisites

Before you begin, ensure you have:

<CardGroup cols={2}>
  <Card title="Institution Verification" icon="building">
    Your institution must be a licensed financial entity with proper regulatory credentials
  </Card>
  <Card title="KYB Completion" icon="id-card">
    Complete Know-Your-Business screening for network participation
  </Card>
  <Card title="Node Infrastructure" icon="server">
    Infrastructure to run a validator node or API integration capability
  </Card>
  <Card title="Development Environment" icon="code">
    Node.js 18+ or Python 3.9+ for SDK integration
  </Card>
</CardGroup>

## Step 1: Apply for Network Access

<Steps>
  <Step title="Submit Application">
    Contact the Meridian team to begin the onboarding process:
    - Provide corporate registration documents
    - Submit regulatory licenses
    - Complete beneficial ownership disclosure
    - Describe intended use cases
  </Step>

  <Step title="KYB Verification">
    Complete the comprehensive know-your-business screening:
    - Legal entity verification
    - License validation
    - Sanctions screening
    - Adverse media check
  </Step>

  <Step title="Technical Onboarding">
    Work with the Meridian team to set up your infrastructure:
    - Validator node setup (if becoming a validator)
    - API credentials provisioning
    - Network connectivity testing
    - Security configuration
  </Step>

  <Step title="Testnet Access">
    Receive access to the Meridian testnet to:
    - Test integrations safely
    - Validate compliance workflows
    - Train your team
    - Prepare for production
  </Step>
</Steps>

## Step 3: Initialize Your Connection

<CodeGroup>
```javascript JavaScript
import { Meridian } from '@meridian/sdk';

const meridian = new Meridian({
  network: 'testnet', // or 'mainnet'
  apiKey: process.env.MERIDIAN_API_KEY,
  privateKey: process.env.MERIDIAN_PRIVATE_KEY,
  institutionId: 'your-institution-id'
});

// Verify connection
const status = await meridian.getNetworkStatus();
console.log('Connected to Meridian:', status);
```

```python Python
from meridian import Meridian

meridian = Meridian(
    network='testnet',  # or 'mainnet'
    api_key=os.getenv('MERIDIAN_API_KEY'),
    private_key=os.getenv('MERIDIAN_PRIVATE_KEY'),
    institution_id='your-institution-id'
)

# Verify connection
status = meridian.get_network_status()
print('Connected to Meridian:', status)
```
</CodeGroup>

## Step 4: Register Your Institution

Register your institution in the RADIUS registry to enable discovery:

<CodeGroup>
```javascript JavaScript
// Register your institution with service offerings
const registration = await meridian.registry.register({
  name: 'Your Bank Name',
  jurisdiction: 'US',
  licenses: [
    {
      type: 'MSB',
      number: '31000-xxx-xxx',
      issuer: 'FinCEN',
      expiryDate: '2026-12-31'
    }
  ],
  services: [
    'remittance-receiving',
    'usd-offramp'
  ],
  supportedCurrencies: ['USD', 'EUR', 'NGN'],
  contactEndpoint: 'https://api.yourbank.com/meridian',
  verifiableCredentials: [
    // Your KYB credentials
  ],
  optInForDiscovery: true
});

console.log('Registration complete:', registration);
```


## Step 5: Complete Your First Settlement

### For Remittance Senders

<CodeGroup>
```javascript JavaScript
// 1. Discover receiving institution
const receivers = await meridian.registry.query({
  jurisdiction: 'NG',
  supportedCurrencies: ['NGN'],
  services: ['remittance-receiving']
});

const receiver = receivers[0];

// 2. Verify credentials
const valid = await meridian.verifyCredentials(
  receiver.verifiableCredentials
);

if (!valid) throw new Error('Invalid credentials');

// 3. Initiate RADIUS handshake with Travel Rule data
const handshake = await meridian.radius.initiateHandshake({
  receiver: receiver.institutionId,
  originator: {
    name: 'John Doe',
    accountNumber: '123456789',
    address: '123 Main St, New York, NY 10001'
  },
  beneficiary: {
    name: 'Jane Smith',
    accountNumber: '987654321'
  },
  amount: '1000.00',
  currency: 'USD'
});

// 4. Wait for receiver acceptance
await handshake.waitForSignature();

// 5. Execute settlement
const tx = await meridian.settle({
  handshakeId: handshake.id,
  amount: '1000',
  token: 'USDC'
});

// 6. Confirm finality (~1-2 seconds)
await tx.waitForFinality();
console.log('Settlement complete:', tx.hash);
```

```python Python
# 1. Discover receiving institution
receivers = meridian.registry.query(
    jurisdiction='NG',
    supported_currencies=['NGN'],
    services=['remittance-receiving']
)

receiver = receivers[0]

# 2. Verify credentials
valid = meridian.verify_credentials(
    receiver['verifiable_credentials']
)

if not valid:
    raise Exception('Invalid credentials')

handshake = meridian.radius.initiate_handshake(

        'name': 'John Doe',
        'account_number': '123456789',
        'address': '123 Main St, New York, NY 10001'
    },
    beneficiary={
        'name': 'Jane Smith',
        'account_number': '987654321'
    },
    amount='1000.00',
    currency='USD'
)

# 4. Wait for receiver acceptance
handshake.wait_for_signature()

# 5. Execute settlement
tx = meridian.settle(
    handshake_id=handshake.id,
    amount='1000',
    token='USDC'
)

# 6. Confirm finality (~1-2 seconds)
tx.wait_for_finality()
print('Settlement complete:', tx.hash)
```

### For Remittance Receivers

<CodeGroup>
```javascript JavaScript
// Set up webhook endpoint to receive handshake requests
meridian.radius.onHandshakeRequest(async (handshake) => {
  // 1. Verify originator institution credentials
  const senderValid = await meridian.verifyCredentials(
    handshake.sender.credentials
  );

  if (!senderValid) {
    return handshake.reject('Invalid sender credentials');
  }

  // 2. Perform compliance checks on Travel Rule data
  const complianceCheck = await performComplianceChecks({
    originator: handshake.originator,
    beneficiary: handshake.beneficiary,
    amount: handshake.amount
  });

  if (!complianceCheck.passed) {
    return handshake.reject('Compliance check failed');
  }

  // 3. Verify beneficiary exists in your system
  const beneficiary = await lookupAccount(
    handshake.beneficiary.accountNumber
  );

  if (!beneficiary) {
    return handshake.reject('Beneficiary not found');
  }

  // 4. Accept and sign handshake
  await handshake.accept({
    beneficiary: {
      name: beneficiary.name,
      accountNumber: beneficiary.accountNumber
    }
  });

  // 5. Listen for settlement
  handshake.onSettlement(async (tx) => {
    // Credit beneficiary account in local currency
    await creditAccount(
      beneficiary.accountNumber,
      convertToLocalCurrency(tx.amount, 'NGN')
    );

    console.log('Beneficiary credited:', tx.hash);
  });
});
```

```python Python
# Set up webhook endpoint to receive handshake requests
@meridian.radius.on_handshake_request
async def handle_handshake(handshake):
    # 1. Verify originator institution credentials
    sender_valid = await meridian.verify_credentials(
        handshake.sender.credentials
    )

    if not sender_valid:
        return handshake.reject('Invalid sender credentials')

    # 2. Perform compliance checks on Travel Rule data
    compliance_check = await perform_compliance_checks(
        originator=handshake.originator,
        beneficiary=handshake.beneficiary,
        amount=handshake.amount
    )

    if not compliance_check.passed:
        return handshake.reject('Compliance check failed')

    # 3. Verify beneficiary exists in your system
    beneficiary = await lookup_account(
        handshake.beneficiary.account_number
    )

    if not beneficiary:
        return handshake.reject('Beneficiary not found')

    # 4. Accept and sign handshake
    await handshake.accept(
        beneficiary={
            'name': beneficiary.name,
            'account_number': beneficiary.account_number
        }
    )

    # 5. Listen for settlement
    @handshake.on_settlement
    async def handle_settlement(tx):
        # Credit beneficiary account in local currency
        await credit_account(
            beneficiary.account_number,
            convert_to_local_currency(tx.amount, 'NGN')
        )

        print('Beneficiary credited:', tx.hash)
```
